
# Road Accident Analysis Dashboard (Power BI)

## Project Overview
This project analyzes road accident data using Power BI to identify patterns in casualties, accident severity, road types, and monthly trends.

## Tools Used
- Power BI
- Excel
- Data Visualization

## Features
- Total casualties analysis
- Fatal, serious and slight casualties
- Monthly accident trends
- Road type analysis
- Urban vs rural accident comparison
- Interactive dashboard

## Dataset
Road Accident Dataset containing accident details, road type, casualty numbers, and location data.

## Output
Power BI dashboard visualizing accident statistics.
